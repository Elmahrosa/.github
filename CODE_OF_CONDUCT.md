# Code of Conduct

Elmahrosa repositories are built to support civic-first, sovereignty-aligned digital infrastructure.

## Our Standards
We expect:
- Respectful, professional collaboration
- Constructive, evidence-based technical discussions
- Security-first and compliance-minded implementation
- No harassment, hate speech, or discriminatory behavior

## Unacceptable Behavior
- Personal attacks, threats, intimidation
- Doxxing or sharing private information
- Deliberate sabotage, backdoors, or malicious contributions
- Any conduct that undermines user safety or repository integrity

## Enforcement
Project maintainers may remove, edit, or reject contributions and may restrict participation for violations.

## Reporting
Report issues responsibly:
- Security issues: follow `SECURITY.md`
- Conduct issues: open a private report via the contact method listed in `SECURITY.md`
